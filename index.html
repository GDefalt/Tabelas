Tags e suas funcionalidades
Está com duvidas sobre tags e suas funcionalidades? Então de uma conferida em nossas tabelas !!
HTML
Tag Descrição
<html> Define o início e o fim do documento HTML.
<head> Contém informações sobre o documento.
<title> Define o título do documento.
<meta> Fornece metadados sobre o documento.
<body> Contém o conteúdo visível da página.
<h1> até <h6> Define títulos de diferentes níveis de importância.
<p> Define um parágrafo.
<a> Define um hiperlink.
<ul> e <ol> Define listas não ordenadas e ordenadas.
<li> Define um item de lista.
CSS
Seletor Descrição
selector { } Define um conjunto de regras de estilo.
.class { } Aplica estilo a elementos com uma classe específica.
#id { } Aplica estilo a um elemento com um ID específico.
property: value; Define uma propriedade de estilo e seu valor.
font-family Especifica a fonte do texto.
color Especifica a cor do texto.
background-color Especifica a cor de fundo do elemento.
margin Define a margem ao redor do elemento.
padding Define o espaço interno do elemento.
display Especifica como o elemento é exibido.
JavaScript
Método/Elemento Descrição
document.getElementById() Retorna uma referência ao elemento por meio do ID.
document.querySelector() Retorna o primeiro elemento que corresponde a um seletor CSS especificado.
addEventListener() Associa um evento a um elemento do DOM.
setTimeout() Executa uma função após um certo tempo definido em milissegundos.
setInterval() Executa uma função em intervalos regulares de tempo.
if / else Estrutura condicional para tomar decisões.
for Loop que executa um bloco de código um número especificado de vezes.
function Define uma função JavaScript.
array.push() Adiciona um ou mais elementos ao final de um array e retorna o novo comprimento do array.
array.pop() Remove o último elemento de um array e o retorna.
PHP
Sintaxe/Elemento Descrição
<?php ... ?> Delimita código PHP dentro do HTML.
echo Exibe texto ou variáveis no navegador.
print Exibe texto no navegador.
$variavel Declaração de uma variável PHP.
if / else Estrutura condicional para tomar decisões.
switch Seleciona um dos muitos blocos de código a serem executados.
while Cria um loop que executa um bloco de código enquanto uma condição específica for verdadeira.
for Cria um loop que executa um bloco de código um número especificado de vezes.
function Define uma função PHP.
include Insere e executa o conteúdo de um arquivo.
require Insere e executa o conteúdo de um arquivo, mas gera um erro fatal se o arquivo não puder ser incluído.
